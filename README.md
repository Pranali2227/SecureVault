# SecureVault - Behavioral Biometrics Authentication

A sophisticated behavioral biometrics authentication system with keystroke dynamics analysis.

## Features
- 🔐 Behavioral biometrics authentication (keystroke dynamics)
- 📊 Real-time biometric analysis
- 🎯 3-attempt intrusion detection
- 📱 Telegram alerts on unauthorized access
- 💾 Persistent user data storage
- 🌓 Dark/Light theme support

## How to Use
1. Visit: https://pranali2227.github.io/SecureVault/
2. Create account with Gmail & password
3. Complete 3 calibration attempts (type your password)
4. Authenticate using your unique typing pattern

## Project Info
- Technology: Vanilla JavaScript, HTML5, CSS3
- Storage: Browser localStorage
- Alerts: Telegram Bot API
- Deployment: GitHub Pages
